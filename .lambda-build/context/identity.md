# identity.md — Professional Identity (AI Engineering + Enterprise Automation)

## Core Identity
**AI Engineer / Automation Builder / Systems Professional** who combines:
- AI engineering (prompting, LLM apps, evaluation/auditing)
- agentic workflow design (tool-using agents, orchestration)
- enterprise automation (n8n + integrations)
- software/systems delivery
- business analysis + technical writing (requirements → docs → rollout)
